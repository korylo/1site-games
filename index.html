loja de Console antigos

Nossa loja tem conseito de trazer os Games que marcarao epoca nos anos 90,nem todos tinha condiçoes de ter um console ai surgiu os games alugado. tinha de varias formas de vc alugar os cartuchos, e tambem os cosole e ficava ate uma semana com equipamento e tambem tinha como vc alugar horas vc jogava quantas horas vc quisece so dependia do seu dinheiro.

as lendas dos game sao:SUPERNITENDO,MEGA TRIVE,ATARI.sao esses 3 que deu esse impacto no mundo dos games cada ano foi passando foi evoluindo cada vez mais ate chegar onde estamos hoje com jogos praticamente quase real,esse mundo nao para de crecer cada vez mais evoluindo em nossas vidas estamos aqui para atendelos em qualquer modelo de console.
